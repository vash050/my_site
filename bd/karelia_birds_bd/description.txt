Данная база данных содержит данные по птицам Карелии.
Включает в себя:
 описание,
 питание,
 фото(в данный момент эти данные сгенерированны), 
 видео(в данный момент эти данные сгенерированны),
 аудио(в данный момент эти данные сгенерированны),
 и т.д.
Позволяет делать выборки по отрядам, видам, фото, видео и т.д.
Вдальнейшем планирую использовать данную БД для создания сайта.
(в данный момент внесена малая часть данных)
(в готовом виде в БД будет содержаться описание ~200 видов птиц.)
